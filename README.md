# Natural Gas Price Forecasting System

A Machine Learning and Data Analytics project that analyzes historical natural gas prices and predicts future prices using Python.

## Features

- Historical price analysis
- Future price forecasting
- Date-wise price estimation
- Data visualization using graphs
- Forecast export to CSV

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- SciPy
- Scikit-learn

## Machine Learning Used

- Linear Regression
- Interpolation
- Time Series Forecasting

## Run Project

```bash
pip install -r requirements.txt
python forecast.py
```

## Output

- Historical price graph
- Forecast graph
- Future forecast CSV
- Price prediction for any date
